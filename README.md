# Weather Dashboard

A front end weather app where users can search for any city worldwide and view current weather, temperature, conditions, location, and a 5-day forecast

## Features

- Search worldwide cities
- View current temperature
- View weather conditions
- View feels-like temperature
- View humidity and wind speed
- View 5 day forecast
- Saves the last searched city with local storage
- Responsive design for mobile and desktop

## Built With

- HTML
- CSS
- JavaScript
- Open Meteo API
- Local Storage

## Project Goal

This project was built as part of my front end developer portfolio to practice API fetching, DOM manipulation, form handling, local storage, and responsive design
