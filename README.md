# Weather Dashboard

A front end weather app where users can search for any city or location worldwide and view current weather, temperature, conditions, location, and a 5-day forecast.

## Live Demo

https://jfoster294.github.io/weather-dashboard/

## Features

- Search worldwide cities and locations
- View current temperature
- View weather conditions
- View feels-like temperature
- View humidity and wind speed
- View 5-day forecast
- Autocomplete search suggestions
- Theme selector with multiple visual styles
- Saves the selected theme with local storage
- Saves the last selected location with local storage
- Responsive design for mobile and desktop

## Theme Options

- Glass Storm
- Cyber Terminal
- Apple Clean
- Weather Mood
- Travel Passport
- Black Luxury
- Nature Calm

## Built With

- HTML
- CSS
- JavaScript
- Open-Meteo API
- Local Storage

## Project Goal

This project was built as part of my front-end developer portfolio to practice API fetching, DOM manipulation, form handling, autocomplete search, theme switching, local storage, and responsive design.
